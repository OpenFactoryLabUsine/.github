# OpenFactory Lab-Usine
Cette organisation regroupe tous les projets en lien avec OpenFactory au Lab-Usine.

OpenFactory est une plateforme développée par Rolf Wuthrich, de l'Université de Concordia. Pour plus d'informations détaillées sur son architecture ou autres, voir l'organisation [openfactoryio](https://github.com/openfactoryio).
